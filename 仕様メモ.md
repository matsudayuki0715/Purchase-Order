# 発注書作成ツール 仕様メモ

## ファイル構成

```
creating quotation_claude/
├── main.py              メインアプリ（GUI + 処理ロジック）
├── template.xlsx        発注書テンプレート（既存フォーマットをベース）
├── 発注書作成ツール.bat  起動用バッチ
├── requirements.txt     必要パッケージ一覧
├── .env                 APIキー等（Git管理外）
├── output/              生成ファイルの出力先（自動作成）
└── 参照/claw_claude/    参考にした見積書GUIプロジェクト
```

## 起動方法

`発注書作成ツール.bat` をダブルクリック

## PDF抽出の処理フロー

```
PDFを選択して「AI抽出」ボタンを押す
  │
  ├─ Step 1: pdfplumber でテキスト抽出
  │           ↓ 30文字未満なら
  │           PyMuPDF でリトライ
  │
  ├─ Step 2: テキストが取れた場合 → 正規表現でパース
  │   ├─ クリエイティング形式: ・名称：xxx  N 個  単価  金額
  │   └─ クリモト形式:         品名  N 式  単価  金額
  │   ↓ 品目が1件も取れなかったら
  │   → エラーメッセージ表示、手動入力で対応
  │
  └─ Step 3: テキスト0文字（画像PDF）→ EasyOCR でOCR → 正規表現でパース
              ↓ それでも失敗したら
              → エラーメッセージ表示、手動入力で対応
```

**コスト：** すべて無料・ローカル動作（APIキー不要）

## 正規表現パターン（`_parse_with_regex`）

| 項目 | パターン | 対応フォーマット |
|---|---|---|
| maker_name | `(株式会社|有限会社)\S+` | KiQ・松田を含む行は除外 |
| quote_no | `見積書?番号\s+([\w-]+)` | クリエイティング等 |
| subject | `件名[：:]\s*(.+)` | クリエイティング |
| subject | `【(.+?見積り?)】` | クリモト |
| delivery | `納期[：:]\s*(受注後[^\n]+)` | クリエイティング |
| delivery | `製作L/T[：:]\s*([^\n]+)` | クリモト |
| items | `・(?:名称\|図番)[：:](.+?)\s+(\d+)\s+(?:個\|式)\s+([\d,]+)` | クリエイティング形式 |
| items | `^(.{3,40}?)\s+(\d+)\s+式\s+([\d,]+)\s+([\d,]+)` | クリモト形式 |
| notes | `【見積り?条件】(.+?)` | クリモト |
| notes | `■御見積に関する` 以降 | クリエイティング |

## Excelテンプレートのセルマッピング（設定シート）

| フォーム項目 | セル | 備考 |
|---|---|---|
| 宛先（会社名） | B2 | |
| 〃（担当者） | B3 | 空固定 |
| 〃（敬称） | B4 | 「御中」固定 |
| 発注日（年） | B7 | |
| 発注日（月） | B8 | |
| 発注日（日） | B9 | |
| 連番 | B10 | 発注No.の末尾2桁 |
| 件名 | B13 | |
| 納期 | B14 | |
| 納品場所 | B15 | |
| 支払条件 | B16 | |
| 見積No. | B17 | |
| 備考 | J2 | 発注書シートE83に表示される |
| 明細・品名 | E列（行2〜26） | |
| 明細・数量 | F列（行2〜26） | |
| 明細・単価 | G列（行2〜26） | 税抜 |

発注書シートの小計・消費税・合計はExcel数式で自動計算される。

## 対応メーカフォーマット実績

| メーカ | PDF種別 | 抽出方法 | 結果 |
|---|---|---|---|
| 株式会社クリエイティング | テキストPDF | pdfplumber + 正規表現 | ✅ 3品目取得 |
| 株式会社クリモト | テキストPDF | pdfplumber + 正規表現 | ✅ 2品目取得 |
| SST設計開発センター | 画像PDF（スキャン） | EasyOCR + 正規表現 | ✅ OCR読取可 |

## 使用ライブラリ

| ライブラリ | 用途 | コスト |
|---|---|---|
| pdfplumber | テキストPDFのテキスト抽出 | 無料 |
| PyMuPDF (fitz) | PDFプレビュー表示 / テキスト抽出補助 | 無料 |
| EasyOCR | 画像PDFのOCR（初回のみ約500MBのモデルDL） | 無料 |
| openpyxl | Excelテンプレートへの書き込み | 無料 |
| pywin32 | ExcelをPDFに変換（win32com経由） | 無料 |
| Pillow | 画像処理補助 | 無料 |
| tkinter | GUI（Python標準） | 無料 |
